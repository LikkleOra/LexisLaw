<task type="auto">
  <name>Task 2: Implement authentication system</name>
  <files>convex/auth.js</files>
  <action>
    Create authentication functions:
    
    1. **register** mutation
       - Accept: { email, password, name, phone }
       - Validate email format and password strength
       - Hash password with bcrypt
       - Create client record
       - Create user record with role: client
       - Return: { success, userId }
    
    2. **login** mutation  
       - Accept: { email, password }
       - Find user by email
       - Compare password hash
       - Generate JWT token (or simple session token)
       - Return: { token, user: { email, name, role } }
    
    3. **getCurrentUser** query
       - Read token from request headers
       - Return user data if valid
    
    Use bcryptjs for hashing (already included in Convex runtime)
  </action>
  <verify>User can register and login successfully</verify>
  <done>Authentication system working with registration and login</done>
</task>

<task type="auto">

  <action>
    Create document management functions:
    
    1. **uploadDocument** mutation
       - Accept: { matterReference, file }

       - Store metadata in documents table
       - Return: { documentId, storageId }
    
    2. **getDocuments** query
       - Accept: { matterReference }

    3. **getDownloadUrl** mutation
       - Accept: { documentId }
       - Generate temporary download URL
       - Return: { url, expiresIn }
    
    File constraints:
    - Max size: 10MB
    - Allowed types: PDF, PNG, JPG, JPEG, DOC, DOCX
  </action>
  <verify>Documents can be uploaded and retrieved</verify>
